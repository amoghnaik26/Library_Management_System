📚 Library Management System
A complete web application for managing library operations including books, members, borrowing, and fines. Built with Flask backend, MySQL database, and responsive HTML/CSS frontend.

✨ Features

Core Features
📊 Interactive Dashboard - Real-time statistics and overview
📖 Book Management - Add, edit, search, and track books
👥 Member Management - Register and manage library members
📅 Borrowing System - Issue and return books with due dates
💰 Fine Management - Automatic fine calculation and payment
📈 Statistics & Reports - View analytics and reports

Additional Features
🔍 Search Functionality - Search books by title or author
📱 Responsive Design - Works on desktop, tablet, and mobile
🎨 Modern UI - Clean and intuitive user interface
⚡ Real-time Updates - Instant data refresh without page reload
🔒 Data Validation - Input validation and error handling

🛠️ Tech Stack

Backend
Framework: Flask 3.1+
Database: MySQL 8.0+
Language: Python 3.8+
Libraries:
flask-cors - Cross-origin resource sharing
mysql-connector-python - MySQL database connector

Frontend
HTML5 - Structure
CSS3 - Styling and animations
JavaScript (ES6) - Interactivity and API calls
Fetch API - Async server communication

Development Tools
Git - Version control
GitHub - Repository hosting
VS Code - Code editor
